# Crypto Client Project

## Running the Application

### Prerequisites

- Java 21
- Maven 3.9.11
- Internet connection (for Bitcoin Testnet API)

### Step 1: Start the Backend

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Step 2: Start the Frontend

```bash
cd frontend
mvn clean install
mvn javafx:run
```
